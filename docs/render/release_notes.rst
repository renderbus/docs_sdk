1.1.0 (2020-04-06):
--------------------

重大更新:

    - rayvision_clarisse 增加独立的本地分析功能,Clarissa分析支持python2

    - rayvision_sync 支持分别上传json配置和资源文件，下载支持自定义服务器文件路径,支持是否使用本地对比功能
